# ABC-BANK-WebSite
A web-based Bank Management System developed as an academic project.

## Technologies Used
-HTML
-CSS
-JavaScript
-XAMPPPHP
-MySQL
-XAMPP

## Features
-User authentication
-Deposit management
-Withdrawal management
-Balance management
-Transaction history

## Installation
1. Clone the repository
2. Import the SQL database
3. Start Apache & MySQL in XAMPP
4. Open:
http://localhost/ABC-BANK-WebSite/
