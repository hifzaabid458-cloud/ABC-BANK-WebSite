# ABC-BANK-WebSite
A web-based Bank Management System developed as an academic project.

## Technologies Used
-HTML
-CSS
-JavaScript
-XAMPPPHP
-MySQL
-XAMPP

## Features
-User authentication
-Deposit management
-Withdrawal management
-Balance management
-Transaction history

## Installation
1. Clone the repository
2. Import the SQL database
3. Start Apache & MySQL in XAMPP
4. Open:
http://localhost/ABC-BANK-WebSite/

## How to Run the Project
1. Install XAMPP.
2. Start Apache and MySQL.
3. Place the project folder inside `C:\xampp\htdocs\`.
4. Create a MySQL database named `abc_bank`.
5. Create the required tables according to the project structure.
6. Open the project in your browser through localhost.
